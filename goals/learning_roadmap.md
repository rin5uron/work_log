## 📚 学習方法

### 基本方針
1. **インプット（30%）**: チュートリアル、ドキュメント読解
2. **アウトプット（70%）**: 実際にコードを書いて実装
3. **振り返り**: 学んだことをwork_log.mdに記録

### 日次ルーティン
1. **15分**: 前日の復習
2. **45分**: 新しい概念の学習
3. **60分**: 実践・コーディング
4. **15分**: 振り返りと記録

### 学習リソース
- **公式ドキュメント**: MDN Web Docs
- **チュートリアル**: freeCodeCamp, Udemy
- **参考書**: 「HTML&CSSとWebデザイン入門講座」
- **実践**: 自分のプロジェクト開発
